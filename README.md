Typing Test Web App

A simple and interactive Typing Speed Test web application built using HTML, CSS, and JavaScript. This app helps users improve their typing speed and accuracy in a clean and user-friendly interface.

Features
 Real-time typing speed (WPM) calculation
Accuracy tracking
 Timer-based typing test
 Restart test functionality
 Random paragraph generation
 Responsive design (works on mobile & desktop)
 Technologies Used
HTML5 – Structure of the web app
CSS3 – Styling and layout
JavaScript (Vanilla JS) – Logic and functionality
📂 Project Structure
typing-test/
│── index.html      # Main HTML file
│── style.css       # Styling file
│── script.js       # JavaScript logic
│── README.md       # Project documentation
⚙️ How It Works
A random paragraph is displayed on the screen.
User starts typing in the input box.
Timer starts automatically when typing begins.
The app calculates:
Words Per Minute (WPM)
Typing Accuracy
Results are shown after completion or time ends.
🚀 How to Run the Project

Download or clone the repository:

git clone  https://github.com/Atifkaashif/Simple-Typing-Test-Web-HTML-CSS-JS

Open the project folder:

cd typing-test
Run the project:
Open index.html in your browser
